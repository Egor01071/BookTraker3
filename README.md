# GitHub User Finder 🔍

**Автор:** [Егор Иванво]  
**Дата:** Май 2026

## Описание проекта

**GitHub User Finder** — это desktop-приложение с графическим интерфейсом для поиска пользователей GitHub, добавления их в избранное и сохранения списка избранных локально.

Приложение использует **GitHub REST API v3** для поиска пользователей и получения их аватаров.

## Функциональность

- ✅ Поиск пользователей GitHub по имени (username)
- ✅ Отображение результатов с аватарами, типом аккаунта и ссылкой на профиль
- ✅ Добавление пользователей в избранное
- ✅ Удаление пользователей из избранного
- ✅ Сохранение избранных в JSON-файл (`favorites.json`)
- ✅ Валидация ввода (поле поиска не может быть пустым)
- ✅ Асинхронная загрузка аватаров (интерфейс не зависает)
- ✅ Открытие профиля в браузере по клику на ссылку

## Технологии

| Технология | Назначение |
